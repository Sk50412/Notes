
## operator precedence
1. postfix           ->     exp++    ,    exp--
    prefix             ->   ++exp  ,    --exp
2.  mul/div          ->      *    /    %
3.  add/sub         ->     +   -
4. shift            ->         <<   >>   >>>
5. comparision   ->       <   >   <=    >=  
6. equality           ->    ==     !=
7. bit wise AND  -> &
8. bit wise XOR  -> ^
9. bit wise OR  -> |
10. and -> &&
11. or -> ||
12. conditional -> (condition) ? (true_val) : (false_val)
13. assignment -> =  +=  -= *= /= >>= <<= >>>= &= ^= |=
